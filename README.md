exfat-utils ([v0.4.2](https://github.com/kusumi/exfat-utils/releases/tag/v0.4.2))
========

## About

Rust fork of exfat-utils in [https://github.com/relan/exfat](https://github.com/relan/exfat)

## Supported platforms

Linux / FreeBSD

## Requirements

Rust 1.86.0 or newer

## Build

    $ make

## Install

    $ make install

## Uninstall

    $ make uninstall
